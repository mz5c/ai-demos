# 贪吃蛇游戏玩法说明

## 版本总览

| 版本 | 文件 | 视觉风格 | 玩法 |
|------|------|---------|------|
| v1.0 | `snake.html` | 像素霓虹风 | 经典贪吃蛇 |
| v2.0 | `snake-doodle-v2.html` | 笔记本涂鸦风 | 四模式 + 道具 + 障碍 + 多食物 |

---

# v1.0 · 像素霓虹风

### 基本玩法

- 20×20 棋盘，方向键/WASD 控制蛇移动
- 吃红色方块食物得分 +10，蛇身变长一节
- 撞墙或撞到自己 → 游戏结束
- 速度恒定，不随分数变化

### 操作

| 按键 | 功能 |
|------|------|
| ← ↑ → ↓ / WASD | 控制方向 |
| P / Space | 暂停 / 继续 |
| R | Game Over 后重新开始 |
| Enter | 首次进入游戏开始 |

### 视觉

- 暗色背景 + 霓虹绿蛇身（自头部渐变色）+ 红色脉冲食物
- Press Start 2P 像素字体
- 蛇头眼睛跟随移动方向

# v2.0 · 笔记本涂鸦风（全功能版）

### 视觉

- 暖白牛皮纸背景 + 淡蓝方格线 + 红色左边距线
- 蛇身：深墨水色 + 阴影晕染 + 随机偏移（模拟手绘不完美）
- 食物：五种手绘涂鸦外观（苹果/蓝莓/青柠/葡萄/菠萝）
- 标题手绘波浪下划线，按钮双线手绘边框
- 字体：Kalam（马克笔） + Caveat（手写体）

### 操作

| 按键 | 功能 |
|------|------|
| ← ↑ → ↓ / WASD | 控制方向 |
| Enter | 开始游戏 |
| P / Space | 暂停 / 继续 |
| R | Game Over 后重新开始 |
| Escape / ← 按钮 | 返回标题页（重新选模式） |
| ⚙ 按钮 | 打开设置面板 |

### 四模式选择

开局前显示 2×2 模式卡片，点击选择后按 Enter 或 Start 开始：

#### Classic 经典模式

- 规则：撞墙 / 撞自身 / 撞障碍 → 游戏结束
- 道具和障碍完整启用

#### Time Rush 限时竞速

- 30 秒倒计时
- 吃食物续命：苹果 +1s / 蓝莓 +3s / 青柠 +2s / 葡萄 +3s / 菠萝 +5s
- 倒计时 < 10s 时红色闪烁
- 时间归零 → Time's Up!
- Canvas 圆环进度条

#### Maze 迷宫模式

- 四个手绘迷宫布局随机（Boxes / Spiral / Cross / Honeycomb）
- 撞迷宫墙 → 游戏结束
- 蛇出生在左上角安全区
- 幽灵 buff 无法穿迷宫墙

#### NoWall 穿墙模式

- 永久边界穿墙（撞左出右、撞上出下）
- 撞自身 / 撞障碍 → 游戏结束

### 食物系统

同时最多 **5 个**食物在场，被吃后每 **5 秒**生成一个。

| 食物 | 颜色 | 外观 | 分数 | 效果 |
|------|------|------|------|------|
| 苹果 | 红 #E74C3C | 椭圆果身 + 高光 + 茎 + 叶 | +10 | 普通 |
| 蓝莓 | 蓝 #3498DB | 圆形 + 五角花萼冠 | +15 | 加速 25% 持续 5s（可叠加延长） |
| 青柠 | 绿 #2ECC71 | 圆形 + 切瓣纹理 | +5 | 蛇身缩短 3 节（最少 3 节） |
| 葡萄 | 紫 #9B59B6 | 三颗圆形 + 茎 | +20 | 获得一次边界穿墙机会 |
| 菠萝 | 金 #F1C40F | 椭圆 + 交叉网格 + 叶冠 | +5 | 接下来 3 次得分翻倍（可叠加） |

**Buff 栏** 位于棋盘左侧，显示当前生效的道具和剩余时间/次数，无 buff 时隐藏。

**食物概率** 可通过 ⚙ 设置面板的滑条自由调节（权重制，保存到 localStorage）。

---

### 障碍进化

分数达到阈值时自动出现，撞到即死。

| 分数 | 内容 | 外观 |
|------|------|------|
| ≥ 80 | 2 个静态障碍 | 涂鸦黑块 + 潦草线条 |
| ≥ 160 | 4 个静态 | 同上 |
| ≥ 250 | 6 静态 + 2 移动 | 新增纸飞机（来回移动） |
| ≥ 400 | 8 静态 + 2 快速移动 | 更多障碍 |

---

### 图鉴面板

棋盘右侧（桌面端）或右下角悬浮按钮（移动端）显示：

- 5 种食物说明（颜色 / 名称 / 得分 / 效果）
- 4 级障碍阈值（触发分数 / 数量 / 类型）

---

### 设置面板

点击计分栏 ⚙ 按钮打开：

- 5 个滑条分别控制苹果/蓝莓/青柠/葡萄/菠萝的出现权重（0~100）
- 修改后实时保存到 localStorage，全局生效
- 重置默认一键恢复

### 技术说明

- 纯前端单文件 HTML（CSS + Canvas 2D + Vanilla JS）
- 直接浏览器打开即可玩，无需构建
- 最高分存储于 localStorage
- 响应式：375px ~ 1440px
